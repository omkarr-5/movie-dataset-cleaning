# 🎬 Movie Dataset Cleaning using Python

## 📌 Overview

This project focuses on cleaning and preprocessing a raw movie dataset using Python and Pandas.

The dataset contained:
- Missing values
- Duplicate rows
- Incorrect datatypes
- Inconsistent text formatting
- Placeholder values

The goal was to convert the raw dataset into a clean and analysis-ready format.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 🧹 Cleaning Steps Performed

- Handled missing values
- Removed duplicate records
- Cleaned text columns using `strip()` and formatting methods
- Replaced invalid placeholder values
- Converted columns into proper datatypes
- Extracted release years using regex

---

## 📂 Files Included

- `movies.csv` → Raw dataset
- `Movies_cleaned.csv` → Cleaned dataset
- `Main.ipynb` → Data cleaning notebook
